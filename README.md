# Complaint Management System

This is a full-stack Complaint Management System built using:

- Spring Boot (Backend)
- React (Frontend)
- JWT Authentication
- Microservices Architecture
- Docker

# Features

- User Registration & Login
- Role-based Authentication (Admin/User)
- Complaint Submission & Tracking
- Admin Dashboard for Complaint Management
- Secure APIs using JWT

# Tech Stack

Backend:
- Spring Boot
- JPA/Hibernate
- MySQL
- JWT Security

Frontend:
- React
- Vite
- Context API

Deployment:
- Docker
